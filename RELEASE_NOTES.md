# Release Notes

## v0.1.0 - Initial Public Release

**First public release of Apple Music Lyrics for Windows!**

### Features

- 🎵 Floating lyrics overlay synchronized with Apple Music playback
- 🎨 Customizable appearance (colors, opacity, fonts, glow effects)
- 📐 Multiple display modes (single-line, multi-line, pure mode)
- 🖱️ Click-through mode for transparent interaction
- ⚙️ System tray integration with quick access menu
- 💾 Persistent settings and window position
- 🔄 Automatic synchronization with playback

### What's Included

- Single-file executable (25MB, framework-dependent)
- Requires .NET 10 Runtime ([download here](https://dotnet.microsoft.com/download/dotnet/10.0))
- Windows 10/11 compatible

### Known Limitations

- Windows-only (uses Windows Media Session API)
- Requires Apple Music to be installed and running
- Songs must be played at least once to cache lyrics locally
- Sync accuracy may vary depending on system performance

### Installation

1. Download `AppleMusicLyrics-v0.1.0-win-x64.zip`
2. Extract to any folder
3. Install [.NET 10 Runtime](https://dotnet.microsoft.com/download/dotnet/10.0) if not already installed
4. Run `AppleMusicLyrics.App.exe`
